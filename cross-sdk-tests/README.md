# Cross-SDK Tests

Cross-language receipt verification tests ensuring that receipts signed by one SDK can be verified by every other SDK.

## Test vectors

- `go_vectors.json` — Generated by the Go SDK using the shared keypair from `sdk/py/tests/fixtures/ts_vectors.json`
- `sdk/py/tests/fixtures/ts_vectors.json` — Generated by the TypeScript SDK (canonical source)

Both vector files use the same Ed25519 keypair so any SDK can verify receipts signed by any other.

## Spec example validation

`spec_examples_test.go` validates that the example receipts in `spec/examples/` have correct structure, required fields, and consistent chain properties.

## Running

```sh
# Generate Go vectors (only needed after changing the unsigned receipt or keys)
go run ./cmd/generate-vectors

# Run spec example validation
go test -tags=integration -v

# Each SDK also has its own cross-language tests:
# Go:     cd sdk/go && go test -tags=integration -v -run CrossLanguage
# Python: cd sdk/py && uv run pytest tests/test_cross_language_go.py -v
# TS:     cd sdk/ts && pnpm test -- cross-language
```
